# 이미지·에셋 배치 가이드 (HOMEPLATE)

어떤 파일을 어디에 넣으면, 어떤 화면의 어디에 적용되는지 정리했습니다.  
**이미지는 모두 PNG로 넣을 수 있으며**, 아래 권장 사이즈를 맞추면 표시 품질이 좋습니다.

---

## 1. 디렉터리 구조 (public)

```
public/
├── favicon.ico         # 브라우저 탭 아이콘 (파비콘)
├── brand/              # 브랜드 에셋
├── game-banners/       # 경기 상세 배너 (default.png 또는 경기별)
├── hero/               # 홈 메인 히어로 배너
├── teams/              # 구단 로고 (PNG)
├── assets/stadium/     # 구장 맵 등
└── placeholder/        # 기본 placeholder
```

**파비콘**: 브라우저 탭에 보이는 작은 아이콘입니다. `public/favicon.ico` 파일을 넣으면 자동으로 적용됩니다. ICO 또는 PNG(32×32, 16×16 등)를 사용할 수 있습니다.

---

## 2. 파일별 용도 및 적용 위치

### 2.1 브랜드 / 로고

| 파일 경로                         | 적용 화면 | 적용 위치                                                   | 권장 사이즈 (PNG) |
| --------------------------------- | --------- | ----------------------------------------------------------- | ----------------- |
| `public/brand/homeplate-logo.png` | 전역      | 헤더 로고, 푸터 로고 (로고 이미지로 바꿀 경우 이 경로 사용) | **200×48** 정도   |

**사용 예**:

- 레이아웃 헤더: `src/app/(tabs)/layout.tsx` — 로고 영역에 `<Image src="/brand/homeplate-logo.png" />` 로 교체 가능.
- 푸터: 동일 레이아웃의 Footer 컴포넌트.

**추가**: 새 로고를 주시면 `public/brand/` 에 넣고, 위 컴포넌트에서 경로만 지정하면 됩니다.

---

### 2.2 경기 상세 배너 (game-banners)

| 파일 경로                                        | 적용 화면                  | 적용 위치                                                              | 권장 사이즈 (PNG)                      |
| ------------------------------------------------ | -------------------------- | ---------------------------------------------------------------------- | -------------------------------------- |
| `public/game-banners/default.png`                | 경기 상세 (`/games/[gid]`) | 상단 배너 (경기별 URL 없을 때 **기본**으로 사용)                       | **1200×600** 이상 (가로 비율 2:1 권장) |
| `public/game-banners/경기ID.png` (예: g-100.png) | 경기 상세                  | 상단 배너 (관리자 "배너/포스터 URL"에 `/game-banners/xxx.png` 등록 시) | 동일 **1200×600** 이상                 |

**동작**: 경기별 `bannerUrl`이 있으면 그 URL 사용, 없으면 **`/game-banners/default.png`** 사용.  
경기 상세 전용 기본 이미지를 바꾸려면 **`public/game-banners/default.png`** 파일만 교체하면 됩니다. (홈 히어로와 별개)

---

### 2.3 히어로 / 메인 배너 (홈 상단 단일 배너)

| 파일 경로                                | 적용 화면 | 적용 위치                                                             | 권장 사이즈·포맷                           |
| ---------------------------------------- | --------- | --------------------------------------------------------------------- | ------------------------------------------ |
| **`public/hero/stadium-home.jpg`**       | 홈 (`/`)  | **홈 상단 풀배너 배경** (우선 사용)                                   | **1920×1080** 또는 **2400×1350**, JPG/PNG  |
| `public/hero/hero-1.png`                 | 홈 (`/`)  | 배경 fallback — `stadium-home.jpg` 없거나 로드 실패 시                | **1920×1080** (Full HD) 또는 **2400×1350** |
| `public/hero/hero-2.png`, `hero-3.png` … | (선택)    | 경기 포스터 등 — 경기 `bannerUrl`에 `/hero/xxx.png` 지정 시 사용 가능 | 동일 **1920×1080** 이상                    |

**동작**:

- **홈** 상단은 **단일 히어로**이며, 배경은 **`/hero/stadium-home.jpg`** 를 먼저 사용합니다. (없거나 로드 실패 시 `hero-1.png`로 자동 전환.)
- 경기 포스터/배너는 경기 데이터의 `bannerUrl`(예: `/hero/xxx.png`, `/game-banners/xxx.png`)로 표시됩니다.
- 코드: `src/app/(tabs)/page.tsx` — 배경 `Image`의 `src`가 `stadium-home.jpg` → `onError` 시 `hero-1.png`.

**넣을 이미지 리스트 (홈 배너 갱신용)**:

| 넣을 파일                        | 넣는 위치      | 비고                                                                 |
| -------------------------------- | -------------- | -------------------------------------------------------------------- |
| **stadium-home.jpg** (또는 .png) | `public/hero/` | 홈 상단 풀배너 배경, 다크·레드 테마에 맞는 경기장/야구장 이미지 권장 |
| hero-1.png                       | `public/hero/` | 배경 fallback용                                                      |

**홈 중간 섹션 (BEYOND THE OUTFIELD)**  
해당 구간에 나오는 큰 사진(경기장/프리미엄 체험 분위기)은 아래 파일을 사용합니다.

| 넣을 파일                           | 넣는 위치      | 적용 위치                                           | 권장 사이즈·포맷                 |
| ----------------------------------- | -------------- | --------------------------------------------------- | -------------------------------- |
| **beyond-outfield.png** (또는 .jpg) | `public/hero/` | 홈 **중간** — BEYOND THE OUTFIELD 카드 안 큰 이미지 | **1200×900** (4:3 비율), JPG/PNG |

- 없거나 로드 실패 시 `hero-1.png`로 자동 전환됩니다.

**정리**: **홈 히어로 배경**은 `public/hero/stadium-home.jpg`, **홈 중간(BEYOND 섹션) 사진**은 `public/hero/beyond-outfield.png` 에 넣으면 적용됩니다.

### 2.5-2 경기장 오는 길 (경기 상세)

| 적용 위치                                    | 설명                                                                                                                         | 권장 사이즈 (PNG)                                |
| -------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ |
| 경기 상세 페이지 **「경기장 오는 길」** 영역 | 현재는 네이버 지도 링크만 표시. 여기에 **지도 이미지**를 넣고 싶다면 코드에서 해당 영역을 `<Image>`로 교체 후 아래 경로 사용 | **1200×400** 또는 **800×400** (가로로 넓은 비율) |

**참고**: 지금은 클릭 시 네이버 지도로 이동하는 링크 박스만 있으며, 영역 높이는 **320px(모바일) / 380px(데스크톱)** 로 키워 두었습니다. 정적 지도 이미지를 쓰려면 `src/app/(tabs)/games/[gid]/page.tsx` 의 「경기장 오는 길」 섹션에서 이미지 경로를 지정하면 됩니다.

**홈 중간(우측) 이미지**: 홈 화면 가운데/오른쪽에 나오는 경기 포스터(VS 이미지)는 **2.3 히어로/메인 배너**와 동일한 소스입니다. 경기 `bannerUrl`에 `/hero/xxx.png` 또는 `/game-banners/xxx.png` 를 넣으면 됩니다. 권장 사이즈는 **1920×1080** 이상입니다.

## 3. 요약 표 (경로 + 권장 사이즈)

| 넣을 파일                | 넣는 위치                                                        | 적용되는 화면·위치                        | 권장 사이즈 (PNG)            |
| ------------------------ | ---------------------------------------------------------------- | ----------------------------------------- | ---------------------------- |
| **서비스 로고**          | `public/brand/homeplate-logo.png`                                | 헤더·푸터 로고                            | **200×48** 정도              |
| **홈 히어로 배경**       | `public/hero/stadium-home.jpg` (우선) / `hero-1.png` (fallback)  | **홈** 상단 풀배너 배경                   | **1920×1080** 이상 (JPG/PNG) |
| **홈 중간(BEYOND) 사진** | `public/hero/beyond-outfield.png` (없으면 `hero-1.png`)          | **홈** BEYOND THE OUTFIELD 섹션 큰 이미지 | **1200×900** (4:3), JPG/PNG  |
| **경기 상세 기본 배너**  | `public/game-banners/default.png`                                | **경기 상세** 상단 (경기별 URL 없을 때)   | **1200×600** 이상            |
| **경기별 포스터**        | 관리자 "배너 URL"에 `/game-banners/xxx.png` 또는 `/hero/xxx.png` | 홈 슬라이더 / 해당 경기 상세 상단         | 위와 동일                    |
| **구단 로고**            | `public/teams/{팀코드}.png`                                      | 홈·일정·경기·티켓(큰 로고)                | **256×256** 또는 **512×512** |
| **구장 맵**              | `public/assets/stadium/jamsil-map.png`                           | 구역 선택 페이지 배경                     | **800×600** 이상             |
| **파비콘**               | `src/app/favicon.ico`                                            | 브라우저 탭                               | 32×32 등                     |

이미지는 **PNG**로 넣으면 되며, 위 경로와 사이즈만 맞추면 해당 화면에 자동 반영됩니다.
