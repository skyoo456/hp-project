# Homeplate Web (프론트엔드)

Next.js App Router 기반 웹 클라이언트.

## 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000) 접속.

## 환경 변수

- **API**: `NEXT_PUBLIC_API_BASE_URL` — 백엔드 API 베이스 (예: `http://localhost:8080`)
- **RUM**: `src/lib/rum/README.md` 참고 (Faro RUM·trace 연동)

## 빌드 / 배포

```bash
npm run build
npm run start
```
